PROYECTO: ¿QUIÉN SOY?  
  

DESCRIPCIÓN DEL PROYECTO:  
Este proyecto titulado “¿Quién soy?” es una página web personal desarrollada utilizando HTML5 y CSS3.  
El objetivo principal es aplicar una estructura semántica adecuada y buenas prácticas de diseño web, mostrando información sobre la autora de forma ordenada y visualmente atractiva.  

El sitio web incluye secciones que presentan información personal, proyectos desarrollados, habilidades técnicas y gustos personales. Además, cuenta con un menú de navegación, un bloque lateral informativo (aside) y un pie de página con enlaces externos a redes sociales.  

El diseño fue elaborado con un enfoque moderno, empleando colores suaves, tipografía legible (Roboto) y efectos de sombra para lograr una apariencia profesional.  

CARACTERÍSTICAS PRINCIPALES:  
- Uso correcto de etiquetas semánticas de HTML5: header, nav, main, section, article, aside, footer, figure y figcaption.  
- Organización clara y jerárquica del contenido.  
- Estilo visual coherente mediante CSS externo.  
- Incorporación de listas, imágenes y enlaces bien estructurados.  
- Presentación de proyectos académicos y personales, como “PlanificaYa” y “SubliTec”.  
- Sección dedicada a gustos personales, artistas favoritos y personas importantes.  

TECNOLOGÍAS UTILIZADAS:  
- HTML5 (estructura del sitio).  
- CSS3 (estilos visuales).  
- Tipografía Roboto (Google Fonts).
